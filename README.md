# Simiso Sibonelo Mthembu — Portfolio

Personal developer portfolio built with vanilla HTML, CSS, and JavaScript. No frameworks, no dependencies — just clean, fast, and responsive.

## 🚀 Live Site

> **[simiso-mthembu.github.io](https://simiso-mthembu.github.io)** *(update this once deployed)*

## 🛠 Stack

- HTML5 · CSS3 · Vanilla JS
- Google Fonts (Inter + JetBrains Mono)
- WhatsApp deep-link contact form

## 📦 Deploying to GitHub Pages

1. Create a new GitHub repo named `<your-github-username>.github.io`
2. Upload `index.html` and `profile.jpg` to the root of the repo
3. Go to **Settings → Pages → Source → Deploy from branch → main**
4. Your site will be live at `https://<your-github-username>.github.io`

## 📁 File Structure

```
/
├── index.html      # Main portfolio page
└── profile.jpg     # Your profile photo (add this yourself)
```

## 📸 Profile Photo

Place your photo in the same folder as `index.html` and name it `profile.jpg`. Recommended size: **400×400px**, square crop.
